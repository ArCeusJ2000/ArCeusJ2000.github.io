参数类型统一

[Error] no matching function for call to 'min(int&, std::basic_string::size_type)'

STL原型

template< class T > 
const T& min( const T& a, const T& b );

报错语句

min(n, str.size());

使用强制类型转换，改为

min(n, int(str.size()));
